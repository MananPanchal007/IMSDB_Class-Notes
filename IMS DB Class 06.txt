  																		1. NO SSA			-> ALL RECORDS TOP-BOTTOM,LET-RIGHT,F-B
																		2. WITH SSA	1. UN-Q-SSA -> ONLY SEGEMENT
																					2. Q-SAA  -> USED FOR GU (BUT -1 TRICK)

																		2. WITH SSA	--> Q-SSA ->  PARTICUALAR RECORD
																				-> ROOT -> 1 SSA
																				-> ROOT + PARENT -> 2 SSA
																				-> ROOT + PARENT + CHILD > 3 SSA

				==> GNP	->  2 STEPS		1. GU CALL -> SET THE PARENT
										2. GNP CALL -> UN-SSA -> ALL RECORD OF THE DATA SEG FOR THAT PARTICULAR RECORD

ISRT ->   USED TO ADD A RECORD TO SEGEMENT**
		-> WE CAN ONLY INSERT 1 SEGEMENT ONLY
		-> ALWAYS THE SEGEMENT  -SSA MUST BE UN-QUALIFIED ONLY

			-> CALL 'CBLTDLI' USING  DLI-ISRT,
								  PCB-MASK,
								  SEGEMENT-IO-AREA,		--> WE WILL INSERT ONLY SINGLE RECORD FOR A SINGLE SEGEMENT ONLY
								  UN-QUAL-SSA.			> POINT TO THE LAST SEGEMENT THAT WE ARE INSERTING INTO
	
	ROOT SEGEMENT -> PART SEGMENT:		
								1. PREPRE MY DATA IN SEGEMENT-IO.			MOVE PART000A  TO KEY-FIELDS , MOVE NAME..
								2. CALL 'CBLTDLI'  USING  DLI-ISRT,
														PCB-MASK,
														SEGEMENT-IO,
														UN-Q-SSA-ROOT.		01 UN-Q-SSA-ROOT   PIC  X(09)  VALUE 'SEGMENT '.


	PARENT SEGEMENT -> PRICE SEGEMENT
									1. PREPARE THE DATA IN PRICE-SEG-IO.
									2. PREPAE THE QUL-SSA FOR THE ROOT  	--> MOVE 'PN00A'  TO KEY-VALUE-ROOT
									3. CALL 'CBLTDLI'  USING DLI-ISRT,
															PCB-MASK,
															PRCIE-SEG-IO,
															QUL-SSA-ROOT,			PN00A
															UN-Q-SSA-PARENT.	PC1,PC2,PC2  (1 BY 1 ONLY)

	CHILD SEGEMENT -> ADDL PRICE SEGEMENT

									1. PREPARE THE DATA IN ADDLSEG-IO.
									2. QUAL-SSA-ROOT SEGMENT			-> MOVE 'PN00A'  TO KEY-VALUE-ROOT
									3. QUAL-SSA-PRICE-SEGEMENT			-> MOVE 'PC1'  TO KEY-VALUE-PARENT
									4. CALL 'CBLTDLI'  USING DLI-ISRT,
														  PCB-MASK,
														  ADDSEG-IO,
														  QUAL-SSA-ROOT,			PN000A
															QUAL-SSA-PARENT,		PC1
															UN-QUAL-SSA-CHILD.		AP1,AP2,AP3 (1 BY 1)

REPL & DELT:
				UPDATE/DELETE 		1.  CALL  'CBLTDLI'  USING DLI-REPL  / DLI-DELT

									2. UNLOAD THE IMS FILE  -> COBOL PGM /SORT  --> LOAD - IMS

	--> CALL - UPDATE OR DELETE  =>   1. LOCK THE RECORD		-> CALL 'CBLDLI'  USING  DLI-GHN/GHU/GHNP

															GHN -> DELETE/UPDATE RECORD SEQ +  HOLD -LOCK THE RECORD
															GHU -> DELTE/UPDATE PARTICULAR  + HOLD
															GHNP -> DELETE/UPDATE -> 	1. GHU -> PARENT
																					2. GHNP -> DELETE  ALL CHILDERN
								  2. UPDATE / DELETE


		HOW TO SET A PARENTAGE  -->  1. GU -> SEGEMENT
								  2. GNP -> CHILD OF THAT SEGEMENT ONLY


		IN BETWEEN  -> HOLD CALL *GHU,GHN,GHNP  &&&  DELETE/UPDATE CALL -> THERE SHOULD NO OTHER CALL ON THE DATABASE


			GHU --- REPL  = VALID
			GHU --- DELE  -> VALID

			GHU -- GU -- REPL -> INVALID  -> IX
			GHU --GN -- DELE --> INVALID
